<h1>Simple Pomodoro Timer</h1>
<hr/>
<h2>Features</h2>
- customizable intervals: set your own study and break durations,
- progress visualization,
- manual mode toggle,
- automatic mode swap: closing the session-end popup automatically toggles the mode and starts the timer,
- audio alerts: notifications when sessions end with volume setting,
- goal tracking: set your study session goal.
